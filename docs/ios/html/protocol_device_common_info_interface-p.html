Detailed Description

This protocol is created to make an API for the devices so that similar operations can be performed. All of the devices are CBPeripheral devices. They have different UUIDs for their services and differing characteristics, so their interactions are different. This API is an attempt to hide the complexity for the sake of brevity in the code and to make things more readable elsewhere. Device specific operations are done in the class files associated with the devices.
