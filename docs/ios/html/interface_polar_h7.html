Detailed Description

This class is for the Polar H7 wristband device. The CBPeripheralDelegate is implemented so that the class can perform all the operations needed to maintain connection status and retrieval of the data. Data retrieveal is dependent on the device type so these methods are defined in this file to account for endianness and data type size as well as possible UUID differences.
