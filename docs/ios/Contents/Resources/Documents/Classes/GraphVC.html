Instance Methods

initWithDevicePoller:

Creates a device selection view controller. The purpose of the method is to create a standard initialization process for the subclasses of this class. All of the subclasses require a bluetooth device manager and need to be active to start building the view. The DevicePollManager functions as the datasource for this viewcontroller.

- (id)initWithDevicePoller:(DevicePollManager *)devicePoller

Parameters

devicePoller

the bluetooth device poller that will serve as the data source.

Return Value

Returns a view controller unless no device manager is given, then the result is nil.
